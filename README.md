# Music-Player
A simple music player that allows users to play, pause, and skip tracks.  Included features like playlist creation and volume control.

## Features
- allows users to play, pause, and skip tracks
- Create Palylist and Control volumn
- Fully Responsive Design
- Easy Installation

## Technologies Used
- WordPress
- PHP
- MySQL
- HTML5
- CSS3
- JavaScript

## How to Use
1. Clone the repository:
   ```bash
   git clone https://github.com/riyankasarkar/wordpress-project.git
   ```
2. Install WordPress on your local server.
3. Replace the `wp-content` folder with the one from this repository.
4. Import the database via phpMyAdmin.
5. Update the `wp-config.php` file with your database credentials.
